Automata Generator
  • What's a DFA?
  • By Rupa Shankar

Specify the states/transitions of a DFA and an input string to get a visualization of the DFA and the result!

Each header and transition should be on its own line.

Example
DFA states

                s S0 start A 

                t 0, 1 -> S0 

                s S1 

                t 0 -> S1 

                t 1 -> S0
Input
0 1 1




DFA State Grammar
<DFA states> --> <State>+
<State>      --> <Header>                  <Transition>+
<Header>     --> s STATE-NAME start? A?
start - denotes a start state
A - denotes an accepting state (default is rejecting)
<Transition> --> t <Symbols> -> STATE-NAME
<Symbols>    --> SYMBOL, SYMBOL, ...
1+ comma separated symbols
